11. Resumen final fácil

HTML = estructura
CSS = estilo
selector = a quién afecta
propiedad = qué cambia
valor = cómo cambia

margin = espacio por fuera
padding = espacio por dentro
border = borde

flex = ordenar fácil en una dirección
grid = ordenar en filas y columnas
responsive = adaptar a móvil
